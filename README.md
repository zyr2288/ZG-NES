# ZG NES

使用 Typescript 编写的NES模拟器。

大部分代码来自 [tsnes](https://github.com/qpalzmqaz123/tsnes/tree/master)

声音合成代码使用的是 [Blip_Buf](https://code.google.com/archive/p/blip-buf)

以下进行安装测试

```
> npm install
> npm run watch
```

然后运行 `dist/index.html` 即可

主要测试代码在 `src/Test/Index.ts`
